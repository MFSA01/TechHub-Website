# 🚀 TechHub-Website

Selamat datang di **TechHub-Website**! Website modern dan responsif yang dibangun dengan teknologi terkini untuk memberikan pengalaman pengguna terbaik.

---

## ✨ Fitur Utama

- 📱 **Desain Responsif** - Tampil sempurna di semua ukuran perangkat (desktop, tablet, mobile)
- 🎨 **UI/UX Modern** - Interface yang indah dan intuitif menggunakan Bootstrap 5.3
- ⚡ **Performance Optimal** - Website ringan dan cepat dimuat
- 🎯 **User-Friendly** - Navigasi mudah dan intuitif
- 🔄 **Cross-Browser Compatible** - Kompatibel dengan semua browser modern

---

## 🛠️ Teknologi yang Digunakan

| Teknologi | Versi | Deskripsi |
|-----------|-------|-----------|
| **HTML** | 5 | Struktur dan markup halaman web |
| **CSS** | 3 | Styling dan animasi visual |
| **Bootstrap** | 5.3 | Framework CSS untuk desain responsif |

**Komposisi Proyek:**
- HTML: 93.3% (Struktur & Markup)
- CSS: 6.7% (Styling & Utilities)

> 💡 Menggunakan Bootstrap 5.3 untuk mempercepat development dan memastikan konsistensi desain di seluruh platform.

## 📦 Instalasi

### Cara 1: Clone Repository
```bash
git clone https://github.com/MFSA01/TechHub-Website.git
cd TechHub-Website
```

### Cara 2: Buka Langsung di Browser
Cukup buka file `index.html` dengan double-click atau drag ke browser favorit Anda.

### Cara 3: Menggunakan Local Server (Python)
```bash
# Python 3.x
python -m http.server 8000

# Atau Python 2.x
python -m SimpleHTTPServer 8000
```
Kemudian buka browser dan akses `http://localhost:8000`

## 🎯 Cara Penggunaan

1. **Ekstrak atau Clone Repository**
   ```bash
   git clone https://github.com/MFSA01/TechHub-Website.git
   ```

2. **Navigasi ke Folder Proyek**
   ```bash
   cd TechHub-Website
   ```

3. **Buka File `index.html`**
   - Double-click `index.html`, atau
   - Buka dengan text editor untuk mengedit, atau
   - Gunakan local server untuk preview

4. **Edit Sesuai Kebutuhan**
   - Modifikasi HTML untuk mengubah struktur
   - Ubah CSS custom untuk styling tambahan
   - Manfaatkan Bootstrap 5.3 classes untuk layout responsif

5. **Deploy ke Production**
   - Upload ke hosting service (Netlify, Vercel, GitHub Pages, dll)
   - Pastikan semua file tersedia dan path-nya benar
